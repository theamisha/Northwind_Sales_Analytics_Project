📊 Northwind Sales Analytics Dashboard

🔎 Project Overview

This project presents an end-to-end Business Intelligence solution built using the Northwind dataset. The objective is to transform raw transactional data into actionable business insights through data modeling, DAX measures, and interactive Power BI dashboards.

The dashboard provides comprehensive visibility into:

1. Sales performance and revenue trends

2. Customer behavior and segmentation

3. Product and inventory analysis

4. Employee sales performance

5. Shipping and operational efficiency

This project demonstrates practical BI development skills including data preparation, modeling, KPI design, and storytelling through visualization.


🎯 Business Objectives

1. Analyze overall sales trends and category performance

2. Identify high-value customers and regional revenue patterns

3. Evaluate employee contribution to total revenue

4. Monitor inventory levels and reorder risks

5. Assess delivery timelines and shipping performance

6. Enable data-driven decision-making through interactive dashboards
   

🗂 Dataset Description


The Northwind database represents a fictional wholesale company, Northwind Traders, which imports and exports specialty food products globally.

Key Tables Used

1. Customers – Customer details and geographic data

2. Employees – Sales representatives and reporting structure

3. Orders – Order-level transaction data

4. Order Details – Line-level sales data (Fact Table)

5. Products – Product catalog and stock information

6. Suppliers – Vendor information

7. Shippers – Shipping providers

8. Categories – Product classification

The data model follows a star schema structure, with Order Details as the central fact table.

5. Discount Impact
